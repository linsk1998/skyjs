使用方法

var person = {
	name: " 张三 "
};
var proxy = new Proxy(person, {
	set:function(target, property) {
		if(property in target) {
			alert("set:"+property);
			return target[property];
		}else{
			throw "Property \"" + property + "\" does not exist.";
		}
	}
});
proxy.name="李四";

使用限制

  • 只支持get、set
  • target的属性必需先定义
  • 属性名只能用变量名的规则，如person.name="a"支持，person["@#$^%"]="a"不支持
