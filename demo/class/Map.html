Map

对象保存键值对。任何值(对象或者原始值) 都可以作为一个键或一个值。

方法返回描述
set添加元素
has返回布尔值判断是否存在某个元素
clear清除内部所有元素
forEach对数组的每个元素执行一次提供的函数(回调函数)
get返回之前设置的值获取键对应的值，如果不存在，则返回undefined
delete返回是否存在移除 Map 对象中指定的元素
entriesIterator方法返回一个新的包含 [key, value] 对的 Iterator 对象 
//基本用法
var m = new Map();
m.set(1, "black");
m.set(2, "red");
m.set("toString", 2);
m.set({x:1}, 3);

m.forEach(function(value, key, mapObj){
	alert(JSON.stringify(key)+" --> "+JSON.stringify(value));
});
运行 
//删除
var m = new Map([[1,"22"],[2,"33"],[2,"44"],[3,"44"]]);
m['delete'](2);
alert(m.size);//2

m.clear();
alert(m.size);//0
运行 
//转数组
var m = new Map([[1,"22"],[2,"33"],[2,"44"],[3,"44"]]);
alert(JSON.stringify(Array.from(m)));
