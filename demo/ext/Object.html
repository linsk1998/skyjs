Object.keys

方法会返回一个由给定对象的自身可枚举属性组成的数组

var o={a:1,toString:2};
var key=Symbol("key");
o[key]=3;

alert(Object.keys(o));
//a,toString
运行

Object.assign

如果目标对象中的属性具有相同的键，则属性将被源中的属性覆盖。后来的源的属性将类似地覆盖早先的属性

var o1 = { a: 1, b: 1, c: 1 };
var o2 = { b: 2, c: 2 };
var o3 = { c: 3 };

var obj = Object.assign( o1, o2, o3);
alert(o1===obj);
alert(JSON.stringify(o1));  // { a: 1, b: 2, c: 3 }, 注意目标对象自身也会改变。
运行
  • 浅拷贝
  • 继承属性不拷贝，建议使用PlainObject
