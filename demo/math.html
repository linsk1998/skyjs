数学函数

Sky.pad

开头补0，字符串补全使用String.prototype.padStart。

Sky.pad(5,4)

Sky.nextSequence

获取下一个自增序列。

Sky.UUID

生成一个随机UUID，返回一个Promise。

Sky.round

四舍五入

Sky.round(4.006);
// => 4

Sky.round(4.006, 2);
// => 4.01

Sky.round(4060, -2);
// => 4100

Sky.ceil

进一法

Sky.ceil(4.006);
// => 5

Sky.ceil(6.004, 2);
// => 6.01

Sky.ceil(6040, -2);
// => 6100

Sky.floor

去尾法

Sky.floor(4.006);
// => 4

Sky.floor(0.046, 2);
// => 0.04

Sky.floor(4060, -2);
// => 4000

Sky.random

整数随机数

Sky.random(2,5);
// => 2 3 4 5
