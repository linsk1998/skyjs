扩展对象

Array

Array.prototype.indexOf

返回给定元素能找在数组中找到的第一个索引值,否则返回-1。

var a = [2, 9, 9];
a.indexOf(2); // 0
a.indexOf(7); // -1
if (a.indexOf(7) === -1) {
	// element doesn't exist in array
}

Array.prototype.lastIndexOf

返回指定元素在数组中的最后一个的索引，如果不存在则返回 -1。从数组的后面向前查找，从 fromIndex 处开始。

Array.prototype.find

方法返回数组中满足提供的测试函数的第一个元素的值。否则返回 undefined。

Array.prototype.findIndex

方法返回数组中满足提供的测试函数的第一个元素的索引。否则返回-1。

function isBigEnough(element) {
	return element >= 15;
}

[12, 5, 8, 130, 44].find(isBigEnough); // 130
[12, 5, 8, 130, 44].findIndex(isBigEnough); // 3

Array.prototype.forEach

对数组的每个元素执行一次提供的函数(回调函数)。

var words=["one","two","three","four"];
words.forEach(function(word){
	console.log(word);
});

Array.prototype.map

返回一个由原数组中的每个元素调用一个指定方法后的返回值组成的新数组。

var numbers = [1, 4, 9];
var roots = numbers.map(Math.sqrt);
/* roots的值为[1, 2, 3], numbers的值仍为[1, 4, 9] */
var users = [{id:"admin", name:"管理员"}, {id:"100360", name:"王小明"}, {id:"100361", name:"张敏敏"}];
var ids = users.map(function(user){
	return user.id;
});
//["admin", "100360", "100361"]

Array.prototype.filter

使用指定的函数测试所有元素，并创建一个包含所有通过测试的元素的新数组。

Array.prototype.some

测试数组中的某些元素是否通过了指定函数的测试。

Array.prototype.every

测试数组的所有元素是否都通过了指定函数的测试。

Array.from

将一个类似数组的对象转化为数组，如arguments、NodeList。

var arr = {'1':1,'length':2};
arr = Array.from(arr);
alert(arr);
//[undefined,1]
Array.from({'1':1,'length':2})

Object.keys

Node

Node.prototype.contains

返回一个布尔值来表示是否传入的节点是该节点的子节点。

String

String.prototype.trim

方法会删除一个字符串两端的空白字符。在这个字符串里的空格包括所有的空格字符 (space, tab, no-break space 等)以及所有的行结束符（如 LF，CR）。

String.prototype.startsWith

用来判断当前字符串是否是以另外一个给定的子字符串“开头”的，根据判断结果返回 true 或 false。

String.prototype.endsWith

用来判断当前字符串是否是以另外一个给定的子字符串“结尾”的，根据判断结果返回 true 或 false。

String.prototype.repeat

构造并返回一个新字符串，该字符串包含被连接在一起的指定数量的字符串的副本。

String.prototype.padStart

会用第二个参数中指定的填充字符串，在当前字符串的头部不断填充，直到它达到第一个参数中指定的目标长度。

'20'.padStart(5,'0')

String.prototype.padEnd

会用第二个参数中指定的填充字符串，在当前字符串的尾部不断填充，直到它达到第一个参数中指定的目标长度。

String.prototype.replaceAll

替换此字符串匹配给定的第一个子字符串。

'\\n\t\d\\n'.replaceAll('\\n','0').replaceAll('\t','1').replaceAll('\d','2')

Function

Function.prototype.bind

bind方法会创建一个新函数,称为绑定函数.当调用这个绑定函数时,绑定函数会以创建它时传入bind方法的第一个参数作为this。
