元素操作

Sky.byId

根据ID获取当前元素

Sky.getCurrentScript

获取当前脚本元素

Sky.getCurrentPath

获取当前脚本的路径

alert(Sky.getCurrentPath());

Sky.getElementStyle

获取元素的样式，包括在CSS中设置的

Sky.getElementStyle(this,prompt('样式名称','color'))
alert(Sky.getElementStyle(this),'color')

Sky.attachEvent detachEvent fireEvent

绑定事件 解除绑定 触发事件

绑定事件 解除绑定 测试按钮

Sky.hasClass addClass removeClass toggleClass

判断是否有一个类样式 添加类样式 移除类样式 切换类样式

Sky.getPrevElement getNextElement

获取前一个元素节点 获取后一个元素节点。排除文本节点之类的。

Sky.getPageTop getPageLeft

获取元素的位置。

Sky.getElementsByClassName

根据一个类样式查找元素，返回一个Array不是HTMLCollection。

Sky.getElementsByClassName(document,"btn")

Sky.getElementsByClassName(this.parentNode, "btn")[1].innerHTML="button" btn

Sky.matches(element, selector)

判断一个元素是否满足id css attribute，匹配成功返回匹配到的元素，不匹配返回false。

alert(Sky.matches(this, ".btn")) alert(Sky.matches(this, ".button"))

Sky.querySelector(element, selector)

多个元素是否满足id css attribute，返回一个Array不是HTMLCollection。

Sky.querySelector(document, ".container-fluid .query-eg .btn,.query-eg")[0].innerHTML="button"
