Reflect

Reflect.apply

通过指定的参数列表发起对目标(target)函数的调用。

Reflect.construct

Reflect.construct() 方法的行为有点像 new 操作符 构造函数 ， 相当于运行 new target(...args)。

Reflect.getPrototypeOf

与 Object.getPrototypeOf() 方法是一样的。都是返回指定对象的原型。

Reflect.defineProperty

定义一个对象的属性。

Reflect.getOwnPropertyDescriptor

获取通过Reflect.defineProperty定义的属性。

Reflect.set

在一个对象上设置一个属性。

Reflect.get

获取通从对象中读取属性。

var o={};
Reflect.defineProperty(o,'name', {
	get: function(){ return "Tom";},
	set: function(value){ alert("setName:"+value);},
	configurable:true
});
alert(Reflect.get(o,'name'));
//Tom

Reflect.set(o,'name',"Tony")
//setName:Tony

Reflect.defineProperty(o,'name', { value: "Mike", configurable:true});
alert(Reflect.get(o,'name'));
//Mike

Reflect.deleteProperty(o,'name');
alert(Reflect.get(o,'name'));
//undefined

var script=Reflect.get(document,'currentScript');
